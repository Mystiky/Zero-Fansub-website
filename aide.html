Le petit dico de l'Otaku
Manga : Bande dessinée japonaise, souvent en noir et blanc et à la pagination élevée.
Mangaka : Auteur ou dessinateur de manga.
Anime : Série télévisée de style manga.
Japon : Pays (archipel) situé à l’extrême est de l’Asie.
Cosplay : Viens de Costume player, ce qui a un rapport avec les costumes et déguisements.
FanArt : L'Art des fans ^^
GoOdies : Produits dérivés ou Bonus.
Fansub
Fansub : Viens de fan subtitles qui signifie sous-titres fait par des fans.
RAW : Version originale sans soutif... sous-titres pardon.
VOST... : Version Originale sous titré. Exemple : Vostfr : sous titres français.
Release : Réalisation.
Trad, Check, time, edit, QC, kara, Enco, MAJeur, Seeder : Phases de l'épisode avant la sortie. + d'infos.
Types de mangas
Shonen : Mangas d'action pour adolescent garçons.
Shojo : Mangas "romantiques" pour adolescentes.
Seinen : Mangas pour adultes.
Doujins : Parodies de mangas par d'autre auteurs que les originaux. /!\Souvent du hentaï.
Hentaï : Mangas à caractéres pornographiques.
Ecchi : Pervers, celui -ci est un intéressé des formes, et petites culottes, contrairement au Hentai qui est un rang de pervers plus sale.
Yaoi : Mangas homosexuelle masculin.
Mots Japonais :
Kawaii : Mignon
Baka : Idiot
Arigato : Merci
Otaku : No-life spécialisés sur les mangas-animes.
Suffixes de prénoms :
-sama : Monsieur, Madame, Seigneur
-dono : Mon Seigneur, mon Prince
-sensei : Maître, professeur
-senpai : Ancien, ainé
-san : Monsieur, Madame, Mademoiselle
-chan : S'emploie principalement envers les femmes, les filles et les jeunes enfants.
-kun : Affection envers quelqu'un de familier, plus pour les garçons, mais peux aussi s'utiliser quand une fille parle à une amie très proche d'elle (rarement employé).
Le guide du téléchargement
DDL :
Vous pouvez télécharger nos releases directement sur notre ftp.
Sur les liens, cliquez à droite et faites "Enregistrer la cible du lien sous...".
MQ : Moyenne qualité.
HQ : Haute qualité.
HD : Très haute qualité.
On peut aussi appeler MQ > LV ; HQ > HD ; FHD > HD.
Torrent :
Pour télécharger avec le systéme de torrent, vous devez posséder un logiciel de torrent.
Sur le tracker torrent de la team, cliquez sur le lien Torrent du fichier que vous voulez télécharger. Faites "Ouvrir avec...votre logiciel".
Le fichier va se télécharger. Plus le nombre est elevé dans la colonne "upload" du tracker, plus le téléchargement sera rapide.
Merci de ne pas déplacer le fichier une fois téléchargé.
